Translate your WordPress site in minutes.

== Description ==
Translate your WordPress into any number of languages with minimal amount of effort.

* Integrate in minutes with minimal effort
* Over 300 languages and dialects
* Use our Universal Translation Memory (UTM) with millions of translations
* Translate your site in-context using our translation tools
* Crowd-source translations to your viewers
* Order professional human translations from our vendors
* Enable new languages with one click of a mouse
* Automatically extract new content and pull in latest translations
* Monitor translation progress and view analytics


== Installation ==

1.  [Sign up for a free Translation Exchange account](https://dashboard.translationexchange.com/signup)
1.  Create a new project and copy your project key.
1.  Install this plugin and open the settings page from the admin menu.
1.  Paste your project key and save the settings.
1.  Add Language Selector widget from "Appearance > Widgets" section.
1.  Visit your site and see it in other languages.

= Minimum Requirements =
* WordPress 3.0 or greater
* PHP version 5.2.4 or greater
* MySQL version 5.0 or greater
